Get your API key

The one key that lets Claude do the work behind your whole system. Five minutes, mostly copy and paste.

💡 What even is this?

Think of it like a power cord. Your Claude account is the brain; the API key is what plugs that brain into your system so it can write your content, replies and follow-ups automatically. You create it once, paste it in, and forget it. Usage is tiny - most clinics spend about $5-15 a month.

1

Open the Anthropic Console

Go to the console and sign in with the same email you used for your Claude account.

console.anthropic.com →
2

Add a payment method

In the left menu, open Billing and add a card. This is usage-based - you only pay for what the system actually uses, which for a clinic is roughly $5-15 a month.

Settings → Billing
Why first? A key created before billing is set up can't actually run anything. Add the card, then make the key.
3

Create the key

Open API Keys, click Create Key, and name it Clinic Catalyst so you know what it is later.

Settings → API Keys → Create Key
4

Copy it now - you only see it once

The key appears one time. Copy it straight into a note, your password manager, or email it to yourself. If you close the window without copying, no harm done - just delete it and make a new one.

Keep it private. Treat it like a password. Don't post it anywhere public - anyone with the key can use your usage.

That's the whole job 🎉

One key, named and saved. Bring it to the workshop and we plug it in together in about thirty seconds.

✅
You're done when: you have a key starting with sk-ant- saved somewhere safe, and a card on file in Billing. That's it - see you on the day.
