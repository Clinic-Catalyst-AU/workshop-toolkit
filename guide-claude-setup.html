💡 What even is this?

Claude is your AI assistant; Claude Code is the small app on your laptop that runs all your Clinic Catalyst skills - the content engine, the follow-up writer, all of it. Your account is the brain, the API key is the power cord that plugs it into your system, and the install puts it on your machine. Usage is tiny - most clinics spend about $5-15 a month.

Before the day · two quick jobs
1

Create your Claude account

Sign up at claude.ai with your clinic email.

Sign up - use this link
2

Make your API key and save it

Go to the Anthropic Console and sign in with the same email as your Claude account. Then, in order:

1. Billing first: in the left menu open Billing and add a card. It is usage-based - you only pay for what the system uses (roughly $5-15 a month for a clinic). A key made before billing is set up can't run anything, so card first.
2. Create the key: open API Keys, click Create Key, and name it Clinic Catalyst.
3. Copy it now - you only see it once. Paste it into a note or your password manager. Closed the window without copying? No harm - delete that key and make a new one.

console.anthropic.com →
Treat it like a password. Anyone with the key can use your usage - keep it private, never post it anywhere.
In the room · we run the install together, first thing
3

On a Mac

Press Cmd+Space, type "Terminal", hit enter - it is just a plain text box, nothing to break. Paste this one line and press enter:

curl -fsSL https://clinic-catalyst-au.github.io/cc-onboard/clinic-bootstrap.sh | bash

That single command sets up Claude Code, all the helper tools and your Clinic Catalyst skills automatically (~25-45 min, mostly downloads - it churns in the background while the workshop runs).

4

On Windows

You already installed Git for Windows in your pre-work (if not, we do it now). We then paste one install line together and it sets up Claude Code and your skills the same way. A couple more steps than Mac - which is exactly why we do it on your screen, together.

5

Start it, sign in, plug in your key

Type claude and press enter. Sign in with your Claude account, and paste the API key you saved when it asks. Done - your skills are on your machine.

claude
