⭐ Kelly's #1 tip for the cleanest install: update your laptop FIRST

The single biggest difference between a 20-minute install and a frustrating one is how up to date your laptop is. On an older system the tools have to be built from scratch instead of downloaded ready-made - and some (like the video skills on Mac) will not run at all until you update. Do this 2-3 days before the workshop, at home, on good wifi, plugged into power.

ON A MAC

1. Check your version: Apple menu  > About This Mac. macOS 14, 15 or 26? You are golden - skip to the checklist below.
2. Seeing 13 or lower? Go to System Settings > General > Software Update and install the newest macOS it offers. Allow 1-2 hours - run it overnight, never the morning of.
3. Make room first: an upgrade wants roughly 25 GB free (About This Mac > Storage). Emptying Downloads and the Bin usually gets you there.
4. Make sure you know your Mac login password - the install will ask for it, and plenty of us have not typed it in years.

ON WINDOWS

1. Start > Settings > Windows Update > Check for updates. Install everything, restart, and repeat until it says "You're up to date".
2. Know your Windows login password too.

EITHER WAY

Keep at least 10 GB of free space for the install itself, restart your laptop on the morning of Day 1, and pack the charger. That is it - an updated machine sails through.
